# 🗣️ Sentiric Tts Coqui Service

Coqui XTTS v2 modeli tabanlı, streaming yeteneklerine sahip Metin-Ses (TTS) sentezleme motoru.

## 🚀 Hızlı Başlangıç
```bash
docker compose up
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **İş Mantığı ve Algoritmalar:** [LOGIC.md](LOGIC.md) dosyasını inceleyin.
* **Anayasal Konum:** [sentiric-spec/spec/services/tts-coqui.spec.yaml](https://github.com/sentiric/sentiric-spec)
